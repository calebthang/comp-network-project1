DNS Concept

DNS (Domain Name System) acts as the internet's phone book, translating human-readable domain names (like google.com) into IP addresses (like 172.217.3.110). This allows users to access websites using memorable names instead of numbers.

HTTP/HTTPS

HTTP/HTTPS handles web traffic, with HTTPS adding encryption for security

IP Addressing

IP Address is a unique numerical identifier assigned to each device on a network. IPv4 uses a 32-bit format (e.g., 192.168.1.1), while IPv6 uses a 128-bit format (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334) to accommodate more devices.
